# UPI-Data-analysis-Using-Power-BI-Power-Querry
The UPI Dashboard is a Power BI report designed to analyze and visualize Unified Payments Interface (UPI) transaction data. It helps users understand transaction trends, volume, and performance metrics across different time periods.

Overview :
- Interactive visualizations for UPI transaction trends.
- Insights on transaction volume and value growth.
- Filters for time period and category analysis.
- Clean, user-friendly dashboard layout.

 Tools Used :
 - Power BI
 - Power Querry
